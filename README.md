# Docs Tasks

Расширение для **Visual Studio Code** и **Cursor**: таблица и канбан по файлам задач `TASK-*.md`, сохранённые виды с фильтрами, редактор YAML **frontmatter** между `---` без изменения основного текста задачи.

**Издатель:** `taraswww777` · **Репозиторий:** [taraswww777/vscode-plugin-docs-tasks](https://github.com/taraswww777/vscode-plugin-docs-tasks)

## Установка

1. Откройте панель расширений и найдите **Docs Tasks** (после публикации в Marketplace).
2. Либо установите файл **`.vsix`**: команда палитры **Extensions: Install from VSIX…** (файл можно взять из [Releases](https://github.com/taraswww777/vscode-plugin-docs-tasks/releases), если они опубликованы).

## Что нужно в проекте

- Откройте папку рабочей области (**File → Open Folder**).
- Файлы задач в каталоге вроде `docs/tasks/` с именами **`TASK-*.md`** и YAML между первыми строками `---`.
- При необходимости путь к каталогу задаётся в настройках (см. ниже).

## Быстрый старт

- **Строка состояния:** слева элемент **«Docs Tasks»** — открывает выбор вида (**таблица** или **канбан**).
- **Проводник:** кнопка с иконкой чеклиста на панели заголовка — то же самое.

Панели открываются **в центральной области редактора**; повторный вызов активирует уже открытую вкладку.

## Возможности

- **Таблица:** фильтры по полям, сортировка по колонкам, **сохранённые виды** (вкладки сверху).
- **Канбан:** колонки по статусу, перетаскивание карточек обновляет `status` и дату **`updated`**.
- **Редактор метаданных** из таблицы/канбана с проверкой перед записью.

## Команды палитры

| Команда | Назначение |
| --- | --- |
| **Docs Tasks: Открыть (таблица или канбан)** | Быстрый выбор вида |
| **Docs Tasks: Таблица (центральная область)** | Только таблица |
| **Docs Tasks: Канбан (центральная область)** | Только канбан |
| **Docs Tasks: Обновить индекс** | Пересканировать задачи |

## Настройки

| Идентификатор | Описание |
| --- | --- |
| **`docs-tasks.tasksFolder`** | Относительный путь к каталогу с `TASK-*.md` для каждого корня workspace (по умолчанию `docs/tasks`). |
